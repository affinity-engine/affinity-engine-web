A `background` fixture extends its definition from the [`image` fixture](#/api/engine/fixtures/images).

#### Example

```js
{
  id: 'bedroom',
  attrs: {
    keyframe: 'bedroom'
  }
}, {
  id: 'forest',
  attrs: {
    layerOrder: [
      'base',
      'snowcover'
    ],
    defaultState: { time: 'day' },
    states: [{
      key: { time: 'day' },
      layers: {
        base: 'forest-day'
      }
    }, {
      key: { time: 'night' },
      layers: {
        base: 'forest-night'
      }
    }, {
      key: { weather: 'snowy' },
      layers: {
        base: 'forest-snow-cover'
      }
    }]
  }
}
```
